# Dubai Mall Interactive Sales Deck

## 📌 Overview
This project is a browser-based interactive sales deck designed to showcase the Dubai Mall as a global retail, entertainment, and lifestyle destination.

The goal is to create an immersive experience for potential tenants, sponsors, and event partners, helping them understand the scale, energy, and business opportunities of the property.

---

## 🚀 Live Project
🔗 https://dubai-mall-deck.netlify.app

---

## 🛠 Tech Stack
- HTML5
- CSS3
- JavaScript
- Netlify (Deployment)

---

## ✨ Features
- Full-screen video-based hero section
- Smooth scrolling navigation
- Scroll-triggered animations
- Interactive hover effects
- Clean and responsive UI
- Section-based storytelling (Retail, Luxury, Dining, Entertainment)

---

## 🎯 Design Decisions
- Used a **minimal, luxury-inspired UI** similar to modern brand websites.
- Focused on **visual storytelling** using images and video.
- Implemented **scroll animations** to enhance engagement.
- Structured content to guide users toward a **business action (CTA)**.
